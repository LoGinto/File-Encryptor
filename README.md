File Encryptor

AES 256 GCM file and folder encryption tool.

This program allows you to encrypt and decrypt files or folders.
Folders are automatically zipped before encryption and automatically unzipped after decryption.

All encryption uses AES 256 GCM.
Each encrypted file has its own unique key.
You choose where to save each key.

If you lose the key, the encrypted file cannot be recovered.

Requirements:
Python 3.8+
